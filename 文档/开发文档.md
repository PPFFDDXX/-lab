前端：arkts

后端：flask+mysql

接口：

1. sign(object O)

   发送路径:```/api/sign```

```json
{
    "is_sign_up":bool
    "user_name":string
    "password":string
}
```

如果失败:帐户不存在：-2，密码错误：-1

成功：0

2. update_user(object O)

   发送路径:```/api/update_user```

   ```json
   {
       "user_name":string
       "password":string
   }
   ```

​	成功：返回成功

3. add_data(Object O)

​	发送路径：```/api/add```

```JSON
{
    "user_name":string
    "type":int
    "money":float
    "content":string
    "date":string//"yyyy-mm-dd"
}
```

失败：未连接数据库500

成功： 200
